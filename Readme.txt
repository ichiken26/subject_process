ダウンロードいただきありがとうございます。製作者の一ノ瀬です。

このアプリケーションの仕様説明を以下に示します。

--------------------------------------------

①参照するCSVのダウンロード
- KOANにログインしてください
-「成績」の「単位修得状況照会」を見てください
- 「表示範囲を指定してください」というメッセージが出ていると思います
- 「過去を含めた全成績」を指定して「CSVに出力する」ボタンをクリックしてください。
- 「指示された条件に該当するデータはxxx件です」というメッセージが出てくるので、「ファイル出力開始」ボタンをクリックしてください
- 文字化けしているので、対象のCSVファイルを選択して右クリック、「プログラムから開く」より「メモ帳」や「テキストエディタ」を選択して開いてください

②CSVファイルの編集
----------
"学生所属コード","学籍番号","画面指定年度","画面指定学期","通算ＧＰＡ","計算日時"	
"学生所属コード","学籍番号","画面指定年度","画面指定学期","年度","１学期ＧＰＡ","１学期計算日時","２学期ＧＰＡ","２学期計算日時"	
受講申請している副専攻・高度副プログラム	
"学生所属コード","学籍番号","画面指定年度","画面指定学期","プログラム区分","プログラム名","コース名","開始年月"
----------

の３項目が上のほうにあると思いますが、これは邪魔者なので消してください。

最終的に１行目が以下のようになっていれば大丈夫です
----------
"学生所属コード","学籍番号","画面指定年度","画面指定学期","No.","科目詳細区分","科目小区分","開講科目名 ","リーディングプログラム科目","知のジムナスティックス科目","単位数","修得年度","修得学期","評語","合否"
----------

このフォルダの中にある「csv」フォルダの中に参考となるテストデータに使ったCSVファイルを入れているのでこれも同様にメモ帳で開いて確認してください。

CSVファイルの編集が完了したら、先ほどの「csv」フォルダの中に「単位取得状況.csv」という名前で保存してください

③アプリケーションの起動・実行
- 「履修状況整理.exe」というファイルがあると思うのでこちらを開いてください
- CSVファイルがなかった場合や形式が異なった場合などはエラーが発生します
- データの整理が問題なく実行された場合、「data exported to "excel/単位取得状況.xlsx"」というポップアップが出てきますので「OK」を押して閉じてください
- 「excel」フォルダの中に整理されたデータが入っていますのでExcelで開いて確認してください
- １つめのシートには、古い順にソートされた単位の取得状況のデータが入っています。
- ２つめのシートには、これまでの取得単位数の合計値が科目の区分ごとに入っています。
